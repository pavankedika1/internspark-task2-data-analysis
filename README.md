Data Analysis and Cleaning Project using Python, Pandas, and Matplotlib. Includes data preprocessing, filtering, grouping, summary statistics, and visualizations on a 10,000-record employee dataset.
# InternSpark Task-2: Data Analysis and Cleaning with Python

## 📌 Project Overview

This project demonstrates data analysis and data cleaning using Python and Pandas. The objective is to load a CSV dataset, clean and preprocess the data, perform filtering and grouping operations, generate summary statistics, and extract meaningful insights through data visualization.

## 🎯 Objectives

* Load a CSV dataset using Pandas
* Perform data cleaning and preprocessing
* Handle missing values and duplicate records
* Filter and group data for analysis
* Generate summary statistics
* Visualize trends and patterns using graphs
* Export cleaned data for future use

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib

## 📂 Dataset Information

The dataset contains **10,000 employee records** with the following attributes:

| Column     | Description         |
| ---------- | ------------------- |
| ID         | Unique Employee ID  |
| Name       | Employee Name       |
| Department | Employee Department |
| Age        | Employee Age        |
| Salary     | Employee Salary     |

## 📊 Analysis Performed

### 1. Data Loading

* Imported CSV file using Pandas
* Displayed dataset overview

### 2. Data Cleaning

* Checked for missing values
* Removed duplicate records
* Ensured data consistency

### 3. Exploratory Data Analysis (EDA)

* Generated descriptive statistics
* Analyzed salary distributions
* Examined department-wise performance

### 4. Data Filtering

* Filtered employees based on salary conditions
* Extracted high-income employee records

### 5. Grouping and Aggregation

* Department-wise average salary
* Maximum and minimum salary analysis
* Average age by department

### 6. Data Visualization

* Bar Chart: Average Salary by Department
* Histogram: Salary Distribution
* Bar Chart: Employee Count by Department
* Scatter Plot: Age vs Salary

## 📈 Key Insights

* Identified department-wise salary trends
* Determined highest and lowest salary ranges
* Analyzed employee age distribution
* Compared employee counts across departments
* Generated actionable insights from employee data
